# center

A cli tool to center text.

## Usage

```bash
Usage:
  center [text]... [flags]

Flags:
  -h, --help      help for center
  -v, --version   version for center
```

## Build

```bash
$ git clone https://github.com/sho7a/center.git
$ cd center
$ make
```
